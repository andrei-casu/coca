Introduction

Coca calendar is a web widget that allows you as page administrator to insert it into your page. In this way, your web page users can have access to their Google Calendar events right on your page. The design is pleasent, and you as a page administrator can choose some parts of it: font-size and font-family, background-colors and many other things.

Structure

Coca Calendar is just in one page built, with buttons that will change the display.

Admininistration Elements

The header of the widget has three main buttons: Preferences, Export and Import

Preferences

When you click Preferences button, the left part of the widget will display a section where you can change your font-size, font-family, color as you like.

You can also see a Google Connect button where you can change the account source for Google Calendar

Export

When you click Export button, the left part of the widget will display a section where you can export the Google Calendar data by donwloading it.

The supported exports are in iCalendar specified format: plain-text (.ics), JSON (jCal), XML (xCal), plus HTML5 (schema.org constructions)

Import

When you click Import button, the left part of the widget will display a section where you can import your own calendar data.

The supported imports are in iCalendar specified format: plain-text (.ics), JSON (jCal), XML (xCal), plus HTML5 (schema.org constructions)

User Elements

The right-side of widget always display a calendar for one month. For each day you can see in the bottom-left corner a small number in red which shows the number of events for that day.

By clicking any of the days, the left-section of the widget will display a list of all events for that day. If you click an event, you can see more informations about that event such as Subject, Location and Additional comments.
